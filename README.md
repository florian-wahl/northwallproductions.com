# NorthWall Productions

Website for NorthWall Productions, an independent film and television production company specializing in northern-set narratives.

## Tech Stack

- **React 18** with React Router v6
- **Vite** for bundling
- **Tailwind CSS** for styling
- **GitHub Pages** for hosting

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run preview` | Preview production build locally |

## Project Structure

```
src/
├── components/   # Reusable UI components (Nav, Footer, SnowCanvas, etc.)
├── data/         # Static data for projects and team members
└── pages/        # Route-level page components
```

## Deployment

The site is deployed to GitHub Pages. Build output goes to `dist/`.
